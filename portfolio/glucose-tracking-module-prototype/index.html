Glucose Charting Drupal Module (prototype)

For Michael Hess’ Content Management Systems course, I built a Drupal module prototype with my friend Sloka Krishnan to handle SMS input of blood glucose levels, intended for customatization and use by people with Type 1 Diabetes (like myself).

The system includes log in capability (only oneself should initially be able to see one’s own glucose levels!), admin interface, date filtering (what if you only want to look at levels over the last two weeks), chart visualization, validation of invalid glucose level input, node creation to support user comments on individual glucose level instances, and functionality that will text the inputter a warning response if blood glucose is unusually high or low.

Other warnings, or alteration of the levels at which warnings occur, is available in the module.

Building the module included exploration of the Drupal 7 Visualization module, the Nexmo API, the Views module, and numerous implementations of Drupal hooks.

The prototype site is available here, though SMS capability is not possible in the prototype without registering one’s own Nexmo account and a Drupal site setup, as we were using the free trial version of the (ultimately very low-cost) Nexmo API.
