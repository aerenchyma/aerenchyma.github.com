Image Classification Project

For our Data Mining course, my friend Aaron Kosel and I decided to investigate different methods of image classification. Using the CalTech 101 categories dataset, a variety of research, and methods ranging from K Nearest Neighbors clustering algorithm to implementation of a neural network, we decided to build a tool people could use. Most of the explanation and description is available on the site; we’re keeping it around because it’s cool, because it was fun, and because we can still add to it. There’s a lot I’d still like to learn about images as feature sets, including digging more into the whys of the convergence we discovered in our limited exploration of neural nets for image classification. Here you can find additional iPython notebooks we worked on in development of the site.
