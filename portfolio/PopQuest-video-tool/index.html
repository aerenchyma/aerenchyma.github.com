PopQuest Tool for Creating Interactive Videos

I was on a team of six (one of two developers) who built a tool for creating interactive (educational) videos using Mozilla’s Popcorn.js framework, at the Summer 2013 MSIS Hack Day (University of Michigan Medical School Information Services).

The project uses client-side JavaScript and Python; more details can be found in the README linked below. I learned a lot about AJAX and client-side development in the process of building the demo.

Still in development, the project is open source under the MIT license and available for tinkering on GitHub here, with a more complete README. A demo is also available.
