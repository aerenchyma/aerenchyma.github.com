Open Education Dashboard: Flask Application

I forked PyDashie, the Python port of the Ruby Dashboard framework Dashing, to create a Flask application to measure Open Educational Resource metrics based on user interivews I’d done to determine what open content creators (such as creators of materials published on Open.Michigan) wanted to discover about their published content in order to measure impact and generate enthusiasm for Open Educational Resources. (With another database or API access, this application is usable for other metrics; its point of access is determined by a file in the directory which should be generated via a cron job.)

This required understanding the structure of the Flask application, including how it differed from Dashing, which is built on top of Sinatra, since I’d worked with the Dashing framework first. I also made a few alterations to the front end of the application, using Rickshaw.js and vanilla CSS.

This application led to the template for the metrics dashboard that Open.Michigan eventually chose for our Dynamic Analytics and Metrics Project, which I am also the lead on, working with my supervisor Kathleen Ludewig Omollo and others in the Open.Michigan Initiative team.
