Open.Michigan REST API

I used the node.js Express framework to throw together a test REST API for Open.Michigan materials. It queries our relational database and returns .JSON responses – of all Open.Michigan courses, all courses with included YouTube videos, all courses created by a certain person or with a certain title… depending upon the request parameters.

This remains in development and is not yet launched. Code can be found on GitHub here.
